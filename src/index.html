Bienvenido al sitio web dónde podrás escribir tu mensaje y obtener un cifrado César, para que hables en claves con tus amigos.

      Instrucciones de uso
      Para Cifrar:
      Elige un desplazamiento (offset), 
      escribe un texto, 
      oprime el botón Cifrar 
      y listo ya tienes tu mensaje.
      Para Descifrar: 
      Copia el texto cifrado, 
      elige el desplazamiento, 
      oprime el botón Descifrar,
      y ya puedes revelar tu mensaje.
      Para borrar oprime el boton resetaer.
